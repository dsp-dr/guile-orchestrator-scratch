* Overview

Guile Orchestrator is a distributed systems orchestration framework implemented in Guile Scheme 3.0+, designed for FreeBSD systems. This project adapts orchestration patterns from modern distributed systems literature into a functional, Scheme-based architecture leveraging Guile's powerful features including Fibers for concurrency, delimited continuations for control flow, and GOOPS for extensible component design.

** Key Features

- Actor-based concurrency model using Guile Fibers
- Declarative orchestration DSL via Guile macros
- Content-addressable task storage
- Distributed coordination with eventual consistency
- FreeBSD-specific optimizations (kqueue, jail integration)
- Literate programming approach with org-mode

* Usage

** Quick Start

#+BEGIN_SRC scheme
(use-modules (orchestrator core)
             (orchestrator actors)
             (orchestrator tasks))

;; Define an actor
(define-actor worker
  ((process task)
   (format #t "Processing ~a~%" task)
   (task-complete! task)))

;; Create orchestrator instance
(define orch (make-orchestrator))

;; Submit a task
(submit-task orch 
  (make-task 'data-processing 
             #:payload '((input . "data.csv")
                        (output . "result.json"))))

;; Start orchestration
(orchestrator-run! orch)
#+END_SRC

** Declarative Orchestration

#+BEGIN_SRC scheme
(define-orchestration data-pipeline
  (stage extract
    #:actors 3
    #:handler extract-data)
  
  (stage transform
    #:actors 5
    #:depends-on (extract)
    #:handler transform-data)
  
  (stage load
    #:actors 2
    #:depends-on (transform)
    #:handler load-data))

(execute-orchestration data-pipeline input-data)
#+END_SRC
